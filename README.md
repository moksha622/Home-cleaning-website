# 🧹 ShinyHomes – Home Cleaning Service Landing Page (React.js)

A modern, fully responsive **Home Cleaning Service Website** built using **React.js, HTML5, CSS3, and JavaScript**.  
This project is designed to help cleaning businesses showcase services, build trust, and increase online bookings.

---

## 🌐 🚀 Live Demo
👉 https://moksha622.github.io/Home-cleaning-website/

---

## 📌 Features

- ✅ Modern Hero Section with Background Image  
- ✅ Fully Responsive Design (Mobile, Tablet, Desktop)  
- ✅ Service Showcase Section  
- ✅ Testimonials Section for Trust Building  
- ✅ About Section  
- ✅ Contact / Booking Form  
- ✅ Clean UI/UX Design  
- ✅ Fast Loading & Optimized Performance  

---

## 🛠️ Technologies Used

- **JavaScript (ES6)**
- **HTML5**
- **CSS3**
- **Flexbox & CSS Grid**
- **Responsive Web Design**
